SmallImage-Smart PNG and JPEG compression

Feature:

Smart PNG and JPEG compression
Optimize your images with a perfect balance in quality and file size.

Uses smart lossy compression techniques to reduce the file size of your PNG files. By selectively decreasing the number of color s in the image, fewer bytes are required to store the data. The effect is nearly invisible but it makes a very large difference in file size!
